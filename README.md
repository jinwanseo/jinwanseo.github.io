## Run Locally

주의: 이 프로젝트는 [Jekyll](https://jekyllrb.com/)을 사용하여 빌드되었습니다. 
로컬에서 실행하려면 `Ruby`와 `Bundler`가 필요합니다. 

```bash
bundle install
bundle exec jekyll serve

// URL 입력, 방문
http://localhost:4000
```

----

### Run Docker

```bash
docker-compose up

// URL 입력, 방문
http://localhost:4000
```
